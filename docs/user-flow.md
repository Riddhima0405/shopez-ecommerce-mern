# 🔄 User Flow – ShopEZ

## 🧑‍💻 Step 1: User Registration
- User visits the platform.
- Registers by creating a new account.
- Provides required details (Name, Email, Password, etc.).
- Account is successfully created.

---

## 🔐 Step 2: User Login
- User logs in using registered credentials.
- Authentication is verified.
- User is redirected to the homepage/dashboard.

---

## 🛍️ Step 3: Browse Products
- User explores available products.
- Views product details including:
  - Description
  - Price
  - Discounts
  - Reviews
- Selects desired product.

---

## 🛒 Step 4: Add to Cart
- User adds selected products to the cart.
- Can:
  - Increase/decrease quantity
  - Remove items from cart
- Cart is linked to the specific User ID.

---

## 💳 Step 5: Checkout Process
- User proceeds to checkout.
- Enters:
  - Shipping address
  - Payment details
- Reviews order summary.
- Confirms the order.

---

## ✅ Step 6: Order Confirmation
- Order is successfully placed.
- Confirmation message is displayed.
- Order details are saved in the database.

---

## 👤 Step 7: View Orders (Profile Section)
- User navigates to Profile.
- Can:
  - View order history
  - Check order details
  - Track order status

---

# 📌 Summary Flow

Register → Login → Browse Products → Add to Cart → Checkout → Order Confirmation → View Orders in Profile
